# axios

vue的http（ajax）请求数据 axios 基于http库一个请求工具

对ajax做了一层简单的封装

手动安装，手动配置

axios基于promise的一个http库。

promise：处理异步方法（函数） 为什么使用promise？ 1.纯函数式的调用 2.链式调用

使用promise的目的？（肯定是为了解决一些已知但是原工具不能友好的解决的问题） 1.回调地狱（函数嵌套，下一层的函数结果需要依赖于上一层函数数据） 2.递归

axios的api的统一管理

跨域问题解决： 1.前端解决 2.后端解决 前端和后台不能同时配置

vue2.x 当前项目下的config文件夹下的index.js

// key自带正则匹配

"/abc": { // 配置项
  	target: 'http://localhost:8099/', // 转发服务器地址--后台服务器地址
  	changeOrigin: true, // 允许跨域
 	pathRewrite: { // 路径重写 // http://localhost:8099/abc/user/getUsers 						http://localhost:8099/apis/user/getUsers
   	 "^/abc": "/apis",
		"^/user": "/user", // 多段路径重写
  	}
   }

注意事项： 1.baseURL 大小写问题 2.拦截器返回值问题（return） 3.api的统一管理（环境切换、export导出、请求方式切换） 4.代理配置 key自带正则匹配 pathRewrite不带正则，手写正则 pathRewrite大小写问题 changeOrigin书写问题
