# unicode方法

在index.html里面引入：

第一步：在 HTML 页面为字体增加 preload，提升字体加载速度：
<link rel="preload" href="//at.alicdn.com/t/font_2722867_2j0nc99ym5c.woff2" as="font" type="font/woff2" crossorigin="anonymous">

在app.vue里面引入：

第二步：拷贝项目下面生成的 @font-face
@font-face {
  font-family: 'iconfont';
  src: url('//at.alicdn.com/t/font_2722867_2j0nc99ym5c.woff2?t=1629425479398') format('woff2'),
       url('//at.alicdn.com/t/font_2722867_2j0nc99ym5c.woff?t=1629425479398') format('woff'),
       url('//at.alicdn.com/t/font_2722867_2j0nc99ym5c.ttf?t=1629425479398') format('truetype');
}

第三步：定义使用 iconfont 的样式
.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

最后在标签里面使用

第四步：挑选相应图标并获取字体编码，应用于页面
<span class="iconfont">&#x33;</span>

# font class的用法

在index.html里面引入：

第一步：在 HTML 页面为字体增加 preload，提升字体加载速度：
<link rel="preload" href="//at.alicdn.com/t/font_2722867_2j0nc99ym5c.woff2" as="font" type="font/woff2" crossorigin="anonymous">

第二步：引入项目下面生成的 fontclass 代码：
<link rel="stylesheet" href="//at.alicdn.com/t/font_2722867_2j0nc99ym5c.css">

最后一步在标签里面使用：

第三步：挑选相应图标并获取类名，应用于页面：<span class="iconfont icon-xxx"></span>

# symbol的用法：

第一步：引入项目下面生成的 symbol 代码：
<script src="//at.alicdn.com/t/font_2722867_2j0nc99ym5c.js"></script>
第二步：加入通用 CSS 代码（引入一次就行）：
<style>
.icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}
</style>
第三步：挑选相应图标并获取类名，应用于页面：
<svg class="icon" aria-hidden="true">
  <use xlink:href="#icon-xxx"></use>
</svg>
